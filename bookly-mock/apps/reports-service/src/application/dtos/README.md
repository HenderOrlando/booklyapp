# Application DTOs - Reports Service

Esta carpeta contiene los Data Transfer Objects (DTOs) de la capa de aplicación.

## Propósito

Los DTOs de aplicación se utilizan para transferir datos entre handlers, servicios y comandos/queries.

## Ejemplos

- `GenerateReportDto` - DTO interno para generar reportes
- `ReportDataDto` - DTO con datos del reporte
- `FeedbackDto` - DTO con información de feedback
- `EvaluationDto` - DTO con datos de evaluación
