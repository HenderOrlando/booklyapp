# Domain Events - Reports Service

Esta carpeta contiene los eventos de dominio del servicio de reportes.

## Propósito

Los eventos de dominio representan hechos que han ocurrido relacionados con reportes y análisis.

## Ejemplos de eventos

- `ReportGeneratedEvent` - Cuando se genera un reporte
- `FeedbackSubmittedEvent` - Cuando se envía feedback
- `EvaluationCreatedEvent` - Cuando se crea una evaluación
- `DashboardRefreshedEvent` - Cuando se actualiza un dashboard
